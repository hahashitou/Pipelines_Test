How To - Sentiment Analysis

Why Cognitive Services, Azure Functions, Azure Logic Apps, and Application Insights

Cognitive services - Text Analytics/Sentiment Analysis

  • Parse unstructured text using Natural Language Processing (NLP) to gain a deeper understanding of customer opinions
  • Identify key phrases, classify terminology, and evaluate across languages

Azure Functions

  • Azure functions allow your teams to write less code, maintain less infrastructure and save on costs.
  • Azure functions are lightweight and stand-alone. Functions can easily break up monolithic applications and reduce your lead/cycle time, allowing your teams to get code to production much more quickly and efficiently.

Azure Logic Apps

  • Logic apps are able to connect to just about anything, with over 200 connectors, and can be triggered at will, on a schedule, or as a response to an event
  • Logic apps are easy to use, with workflows that can be created with just a few configurations using a powerful drag-and-drop editor approach, logic apps can allow your team to quickly and easily orchestrate complex logic

Application Insights

  • Alerting allows for quickly notifying interested parties of an event or a potential service downtime
  • Extensible Application Performance Management (APM) service, used to monitor live applications
  • Automatically detect performance anomalies, leverage powerful analytics tools, and continuously improve performance and usability
