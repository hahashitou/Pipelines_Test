There are different ways to create a function and fullfil this success criteria. If you’re using VSCode and have the Azure Function extension installed, follow following steps:

  1. Ctrl+shift+P to show the palette

  2. In the palette, search for create function app and click

  3. Select HTTP trigger

  4. Enter a name for your function, e.g. bfyoc-your-initials

  5. Provide a namespace, e.g bfyoc.Function

  6. Select access rights Anonymous for now.

A function file should have been created with following code:

using System;
using System.IO;
using System.Threading.Tasks;
using Microsoft.AspNetCore.Mvc;
using Microsoft.Azure.WebJobs;
using Microsoft.Azure.WebJobs.Extensions.Http;
using Microsoft.AspNetCore.Http;
using Microsoft.Extensions.Logging;
using Newtonsoft.Json;

namespace bfyoc.Function
{
    public static class bfyoc_hba
    {
        [FunctionName("bfyoc_hba")]
        public static async Task<IActionResult> Run(
            [HttpTrigger(AuthorizationLevel.Anonymous, "get", "post", Route = null)] HttpRequest req,
            ILogger log)
        {
            log.LogInformation("C# HTTP trigger function processed a request.");

            string name = req.Query["name"];

            string requestBody = await new StreamReader(req.Body).ReadToEndAsync();
            dynamic data = JsonConvert.DeserializeObject(requestBody);
            name = name ?? data?.name;

            string responseMessage = string.IsNullOrEmpty(name)
                ? "This HTTP triggered function executed successfully. Pass a name in the query string or in the request body for a personalized response."
                : $"Hello, {name}. This HTTP triggered function executed successfully.";

            return new OkObjectResult(responseMessage);
        }
    }
}

Success criteria: Everyone on your team must demonstrate that you deployed that HTTP triggered Azure Function to Azure. Show to your coach that you can call that deployed function and receive a successful response.

  1. In VS Code, open the palette with ctrl+shift+P

  2. In the palette search field look for azure functions: deploy to function app

  3. From the list of subscriptions select an Azure subscription

  4. Select + Create new Function App in Azure

  5. Give the new function app a name

  6. Select .NET Core 3.1 as a .NET stack

  7. Select a region in which you want to deploy your finction app

  8. Hit enter, if everything is set correctly, the deloyment should start. After about a minute you should see the notification “Deployment completed” in VS Code

  9. In the Azure portal you should find a new resource group named after your function app

  10. In that resource group navigate to yoru function (name of yoru function app > Functions > your function name), click on Get Function Url and copy the URL of your function

  11. Visit your function’s URL in a new browser app. You should get the following as a response:

  12. This HTTP triggered function executed successfully. Pass a productId in the query string or in the request body for a personalized response.

  13. Add a random productId to the function’s address in the browser’s address; e.g. ?productid=123. You should get the following as a response:

    Hello, 123. This HTTP triggered function executed successfully.
