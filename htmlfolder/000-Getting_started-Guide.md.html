Self-deploy OpenHack

To deploy the Azure resources needed in this OH, follow this step-by-step documentation.

This deployment might take around 5 minutes.

Initial Assets in Azure Portal

  1. Log in to the azure portal

  2. Once in the Azure Portal make sure you have following resource group (the suffix depends of the region you specified while deploying).

If the deployment went successful, you should be able to find following resources inside the created resource group.

The list includes a VM attached to a Network interface controller (NIC). The NIC allows the VM to belong to a virtual network (VNet) and at the same time the NIC holds a public IP of the VM, so it can be accessed from the outside.

In addition to these resources, a network security group (NSG) controls the access to our VM with inbound and outbound security rules.
